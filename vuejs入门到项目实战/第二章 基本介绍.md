### 引入Vuejs(国内镜像)
1. 安装cnpm `npm install -g cnpm --registry=https://registry.npm.taobao.org`
2. 用cnpm安装Vuejs `cnpm install vue`
3. 引入Vue模块 `import Vue from 'vue'`

### Vue实例
- 初始化时，在实例上绑定的常规数据会被Vue转化为**被观察的拥有可响应行为的对象**。简单来说，当数据发生变化时，视图也随之改变。
- 生命周期： Vue在实例化过程中需要经历一系列过程，比如编译模板、渲染虚拟DOM树、将实例挂载到DOM上、设置数据监听和数据绑定等。这些过程中会运行一些钩子函数（hook），允许开发者在不同阶段注入自己的代码。
    1. beforeCreate：实例初始化之后，数据观测和事件绑定之前
    2. created：实例初始化完成，**挂载**（将vue的template挂在DOM节点上）尚未开始时
    3. beforeMount：挂载之前，render函数首次被调用时（render函数是一个用JS写DOM的渲染函数，起源于React，又叫JSX）
    4. mounted：在实例的VDOM挂载到DOM节点上之后
    5. beforeUpdate：数据更新时，在虚拟DOM状态变化之前
    6. updated：数据更新，且虚拟DOM被重新渲染之后
    7. beforeDestroy：实例销毁之前，此时实例依然可用
    8. destroyed：实例销毁后，此时Vue实例及其子实例将完全解绑

### 响应式数据
一方面指衍生数据和元数据之间的响应，通过数据链来实现；一方面指view和数据之间的绑定。
- 数据链：computed能够得到计算属性，以供开发者生成衍生数据对象。虽然计算属性以函数形式声明，但是不接受参数！它能够直接以属性的形式访问到，通过插值表达式{{}}即可。它里面的this指针指向Vue实例，因此能够访问所有的已挂载（已经挂到DOM上的）计算属性。
- 数据绑定视图：defineProperty的方法对对象属性配置。
